1. Abstract

The AI-Based Career Counselling and Aptitude Analysis System represents a significant advancement in educational technology. Built on the MERN stack (MongoDB, Express, React, Node), the platform provides a scientific, data-driven framework for career decision-making. By analyzing student responses to multi-domain cognitive assessments, the system generates a high-fidelity aptitude profile spanning Logical, Analytical, Verbal, and Technical intelligence. These profiles are then processed through a proprietary Weighted-Sum Model (WSM) algorithm to provide objective career recommendations. The system is enhanced with detailed professional roadmaps, context-aware AI interactions, and real-time community engagement, creating an end-to-end guidance ecosystem for the modern student.

2. Introduction

2.1 Project Background

Career selection is a pivotal moment in any student's life, yet it remains one of the least supported phases of education in many parts of the world. In the Indian context, students often rely on traditional advice which may not align with their actual cognitive strengths or the rapidly evolving global job market.

2.2 Problem Statement

Conventional career counseling is frequently expensive, non-scalable, and prone to human bias. Existing digital tools often provide generic recommendations based on superficial interests rather than quantifiable cognitive aptitude. There is a clear need for a tool that uses mathematical modeling to map a student's innate abilities to specific industry roles.

2.3 Proposed Solution

This project proposes an AI-driven SaaS platform that digitizes the career counseling process. By combining psychological assessment principles with modern web technologies, we provide a tool that is not only accessible and free but also scientifically robust. The platform allows students to understand their "Radar Profile" and provides them with a roadmap to reach their top-tier professional matches.

5. Algorithm Deep-Dive

5.1 The Matching Formula

The core of the system is the Weighted-Sum Model (WSM). Each career profile is treated as a 4-dimensional vector where each dimension represents the "importance" of a cognitive domain.

Match % = ∑ (User Scored × Career Weightd)

5.2 Domain Weighting Examples

Career PathLogicalAnalyticalVerbalTechnical
Software Engineer0.300.200.050.45
Lawyer / Advocate0.350.150.450.05
Financial Analyst0.350.450.150.05

This ensures that a student who scores highly in "Verbal" but moderately in "Technical" will see Law and Journalism as top matches, rather than Software Engineering.

const generatePersonalizedReasoning = (careerTitle, normalizedScores) => {
  const dominant = Object.entries(normalizedScores).sort((a, b) => b[1] - a[1])[0][0];
  const score = Math.round(normalizedScores[dominant]);
  const intros = {
    Technical: `You scored ${score}% in Technical skills! This means you are great at computers and building things.`,
    Logical: `With a ${score}% in Logical thinking, you are awesome at solving puzzles and math.`,
    Analytical: `Your ${score}% Analytical score shows you have a detective's mind—finding facts easily.`,
    Verbal: `Achieving ${score}% in Verbal skills means you have excellent communication and reading abilities.`,
  };
  return `${intros[dominant] || ''} Based on your profile, ${careerTitle} is an excellent match.`;
};

exports.getQuestions = async (req, res) => {
  try {
    const [logical, analytical, verbal, technical] = await Promise.all([
      Question.aggregate([{ $match: { category: 'Logical' } }, { $sample: { size: 3 } }]),
      Question.aggregate([{ $match: { category: 'Analytical' } }, { $sample: { size: 3 } }]),
      Question.aggregate([{ $match: { category: 'Verbal' } }, { $sample: { size: 3 } }]),
      Question.aggregate([{ $match: { category: 'Technical' } }, { $sample: { size: 3 } }]),
    ]);
    const questions = [...logical, ...analytical, ...verbal, ...technical].sort(() => Math.random() - 0.5);
    res.json(questions);
  } catch (error) { res.status(500).json({ message: error.message }); }
};

exports.submitTest = async (req, res) => {
  try {
    const { answers } = req.body;
    let rawScores = { Logical: 0, Analytical: 0, Verbal: 0, Technical: 0 };
    let totalPerCat = { Logical: 0, Analytical: 0, Verbal: 0, Technical: 0 };
    const questions = await Question.find({ _id: { $in: Object.keys(answers) } });
    
    for (const q of questions) {
      totalPerCat[q.category] += 1;
      if (q.correctAnswer === answers[q._id.toString()]) rawScores[q.category] += 1;
    }

    const normalizedScores = {
      Logical: totalPerCat.Logical ? (rawScores.Logical / totalPerCat.Logical) * 100 : 0,
      Analytical: totalPerCat.Analytical ? (rawScores.Analytical / totalPerCat.Analytical) * 100 : 0,
      Verbal: totalPerCat.Verbal ? (rawScores.Verbal / totalPerCat.Verbal) * 100 : 0,
      Technical: totalPerCat.Technical ? (rawScores.Technical / totalPerCat.Technical) * 100 : 0,
    };

    const careers = await Career.find({});
    const allScored = careers.map(c => ({
      careerId: c._id,
      career: c.title,
      matchPercentage: Math.round(
        normalizedScores.Logical * (c.weights.Logical || 0) +
        normalizedScores.Analytical * (c.weights.Analytical || 0) +
        normalizedScores.Verbal * (c.weights.Verbal || 0) +
        normalizedScores.Technical * (c.weights.Technical || 0)
      ),
      reasoning: generatePersonalizedReasoning(c.title, normalizedScores)
    }));

    allScored.sort((a, b) => b.matchPercentage - a.matchPercentage);
    const result = await Result.create({
      user: req.user._id,
      scores: rawScores,
      matches: allScored.slice(0, 3),
      recommendedCareers: allScored.slice(0, 3).map(m => m.career)
    });
    res.status(201).json(result);
  } catch (error) { res.status(500).json({ message: error.message }); }
};

exports.getComments = async (req, res) => {
  try {
    const comments = await Comment.find({ careerId: req.params.careerId })
      .populate('user', 'name').sort({ createdAt: -1 });
    res.json(comments);
  } catch (error) { res.status(500).json({ message: error.message }); }

exports.likeComment = async (req, res) => {
  try {
    const comment = await Comment.findById(req.params.id);
    const userId = req.user._id;
    const index = comment.likes.indexOf(userId);
    if (index === -1) comment.likes.push(userId);
    else comment.likes.splice(index, 1);
    await comment.save();
    res.json(comment);
  } catch (error) { res.status(500).json({ message: error.message }); }
};

frontend/src/components/AIChatbot.jsx
import React, { useState } from 'react';
import { motion, AnimatePresence } from 'framer-motion';
import { FiMessageSquare, FiX, FiSend } from 'react-icons/fi';

const buildResponse = (input, resultData) => {
  const lc = input.toLowerCase();
  const testScores = resultData?.scores;
  if (lc.includes('hi') || lc.includes('hello')) {
    if (testScores) return `Hello! Your strongest area is ${Object.entries(testScores).sort((a,b)=>b[1]-a[1])[0][0]}.`;
    return "Hello! I am your AI career assistant. How can I help you today?";
  }
  return "I can help you explore careers. Try asking about Software Engineering or IAS.";
};

const AIChatbot = () => {
  const [isOpen, setIsOpen] = useState(false);
  const [input, setInput] = useState('');
  const [messages, setMessages] = useState([{ text: "👋 Hi! How can I help you today?", isBot: true }]);

  const handleSend = () => {
    if (!input.trim()) return;
    const userMsg = input.trim();
    setMessages(prev => [...prev, { text: userMsg, isBot: false }]);
    setInput('');
    setTimeout(() => {
      const res = buildResponse(userMsg, JSON.parse(localStorage.getItem('lastAptitudeResult')));
      setMessages(prev => [...prev, { text: res, isBot: true }]);
    }, 600);
  };

  return (
    <div className="fixed bottom-6 right-6 z-50">
       <button onClick={() => setIsOpen(!isOpen)} className="p-4 bg-blue-600 text-white rounded-full shadow-2xl hover:scale-110 transition-transform">
        {isOpen ? <FiX size={24} /> : <FiMessageSquare size={24} />}
       </button>
       <AnimatePresence>
        {isOpen && (
          <motion.div initial={{ opacity: 0, scale: 0.8 }} animate={{ opacity: 1, scale: 1 }} exit={{ opacity: 0, scale: 0.8 }} className="absolute bottom-20 right-0 w-80 bg-white dark:bg-gray-900 border border-gray-200 dark:border-gray-800 rounded-3xl shadow-2xl overflow-hidden">
            <div className="p-4 bg-blue-600 text-white font-bold">Career AI</div>
            <div className="h-64 overflow-y-auto p-4 space-y-3">
              {messages.map((m, i) => (
                <div key={i} className={`p-2 rounded-xl text-sm ${m.isBot ? 'bg-gray-100 dark:bg-gray-800 self-start' : 'bg-blue-600 text-white self-end ml-auto w-fit'}`}>{m.text}</div>
              ))}
            </div>
            <div className="p-3 border-t flex gap-2">
              <input value={input} onChange={e => setInput(e.target.value)} onKeyDown={e => e.key === 'Enter' && handleSend()} className="flex-1 text-sm outline-none bg-transparent" placeholder="Type here..." />
              <button onClick={handleSend} className="p-2 text-blue-600"><FiSend /></button>
            </div>
          </motion.div>
        )}
       </AnimatePresence>
    </div>
  );
};
export default AIChatbot;
